# 项目对比

| 项目 | 优点 | 风险 | 建议 |
| --- | --- | --- | --- |
| `bilibili-summary` | 聚焦视频总结，适合知识库和选题生产 | 真实总结需要模型 API；收藏夹功能需要登录 | 作为主力生产工具 |
| `bilibili-ai-bot` | Agent、记忆、人格、主动行为完整 | 有真实评论、点赞、投币、关注、动态等高风险动作 | 只做实验和草稿，不直接跑主循环 |

## 推荐路线

1. 先用 `tools/fetch_bilibili_video_draft.py` 做免登录公开视频草稿。
2. 配好模型 API 后，用 `bilibili-summary` 做真实 AI 总结。
3. `bilibili-ai-bot` 只研究提示词、记忆和草稿生成，暂不接真实账号。
