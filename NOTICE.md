# Notice

This repository is a long-term workspace for Bilibili AI content research and knowledge-base workflows.

It includes local modifications and orchestration around these upstream projects:

- `bilibili-summary`: Bilibili video summarization tool. The upstream README states MIT License.
- `bilibili-ai-bot`: Bilibili AI bot project, licensed under MIT License. Copyright belongs to the original author listed in `bilibili-ai-bot/LICENSE`.

This repository keeps upstream attribution and license information in the corresponding subdirectories. Local additions focus on:

- read-only Bilibili public video discovery,
- public comment sample analysis,
- AI video summary browsing,
- local knowledge-base organization,
- Obsidian export utilities,
- safety wrappers and documentation.

Safety boundary:

- This public repository must not contain API keys, cookies, tokens, passwords, Bilibili login credentials, generated private summaries, subtitle caches, logs, or personal Obsidian content.
- Platform interaction features such as automatic likes, coins, comments, follows, direct messages, and dynamic posts are not enabled by default in this workspace.
- Users should comply with Bilibili terms, copyright rules, and applicable laws.
